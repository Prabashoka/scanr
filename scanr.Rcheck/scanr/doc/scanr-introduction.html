Getting Started with scanr

The scanr package detects change points in univariate time series with window-based SCAN statistics.

library(scanr)

Create a simple series with one mean change around observation 100.

set.seed(123)
x <- c(rnorm(100, mean = 0), rnorm(100, mean = 2))

Run change-point detection with several window sizes.

fit <- scan_cpd(
  x,
  window_sizes = c(20L, 30L),
  n_boot = 100L,
  random_state = 123L,
  change_type = "mean",
  n_jobs = 1L
)

fit
#> scanr change-point result
#> observations: 200 
#> change points: 100
fit$change_points
#> [1] 100

You can compare detected change points against known truth when benchmarking or testing simulated data.

cpd_metrics(
  true_cps = 100L,
  estimated_cps = fit$change_points,
  n = length(x),
  tolerance = 20L
)
#> $matches
#>   true estimated distance
#> 1  100       100        0
#> 
#> $precision
#> [1] 1
#> 
#> $recall
#> [1] 1
#> 
#> $f1
#> [1] 1
#> 
#> $covering
#> [1] 1
